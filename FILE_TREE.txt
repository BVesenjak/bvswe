Animation Details
=================

1. iPad Slide-in (IPadShowcase.vue):
   - Uses Vue transition with Web Animations API
   - translateX(-40px) → 0, opacity 0 → 1
   - Duration: 800ms, cubic-bezier easing

2. iPad Scroll Loop (IPadShowcase.vue):
   - requestAnimationFrame-based state machine
   - States: 'idle' (1000ms) ↔ 'scrolling' (100ms)
   - Scrolls by 50% screen height each cycle
   - Seamlessly loops back to top
   - Cross-fades images during scroll

3. Gallery Auto-swap (GalleryTile.vue):
   - setInterval every 3000ms
   - Cycles through 3 images
   - Stops after 3 swaps
   - CSS opacity transition (500ms)

4. Status Ring (Navbar.vue):
   - CSS keyframes animation
   - Pulses scale + opacity
   - 2s ease-in-out infinite
   - Disabled if prefers-reduced-motion

5. All Transitions:
   - Respect prefers-reduced-motion media query
   - Gracefully degrade to static first frame
   - Keyboard-accessible
   - ARIA-labeled

Build & Deploy
==============

Development:
  npm install
  npm run dev
  → http://localhost:5173

Production Build:
  npm run build
  → Creates `dist/` folder

Preview Build:
  npm run preview
  → Preview production build locally

Deploy to:
  - Netlify (drag & drop `dist/`)
  - Vercel (connect repo or CLI)
  - GitHub Pages (copy `dist/` to gh-pages)
  - Any static host

No server required - fully static site!
